# defect-management

## В рамках задания необходимо выполнить следующее:
### Cоздать монолитное веб-приложение для централизованного управления дефектами на строительных объектах. 
### Система предназначена для:
#### Инженеров (регистрация дефектов, обновление информации);
#### Менеджеров (назначение задач, контроль сроков, формирование отчётов);
#### Руководителей и заказчиков (просмотр прогресса и отчётности).

## Инструкция
### Скачать архив с кодом, распаковать, занести в среду visual studio code. В терминале написать npm install и после установки пакетов написать npm run dev. Запуститься и серверная часть, и клиентская.
